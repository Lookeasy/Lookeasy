#Lookeasy
13/06/26
O Lookeasy é um projeto em desenvolvimento, por 4 estudantes da Etec Bento
Quirino

O site tem como objetivo criar um ambiente integrado a IA, de forma que o usuario mande fotos das suas roupas para o site organizar seu "guarda-roupas virtual" 
A IA vai fazer o controle de roupas sujas e lavadas, quais os melhores dias para se lavar cada grupo de roupas, quais roupas lavar separadamente. Baseado em quando, onde, como, e porque você utiliza cada peça.
O usuario pode mandar mensagens para a IA dizendo onde e quando vai, porque vai, oque quer transmitir no seu visual, etc, e a IA vai montar um look conforme oque o usuario pediu/informou.

Tudo isso com uma assinatura mensal.

O desenvolvimento do Lookeasy começou hoje, 13/06/26.
